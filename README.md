# LifeQuest Planner

A mobile-first gamified calendar, task, gym, and diet planner.

## Free GitHub Pages Deploy

1. Create a new GitHub repository.
2. Upload these project files to the repository root:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `manifest.webmanifest`
   - `.nojekyll`
   - `assets/`
3. In GitHub, open the repository settings.
4. Go to `Pages`.
5. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
6. Save. GitHub will give you a public URL once it finishes deploying.

Your tasks are stored in the browser on the device using the app. That means the hosted app works anywhere, but data does not sync between devices yet.
